== README

Use rails server from the command line to run this app.

Type URL's into browser to run.

URL's that can be run in rails server with this program

http://localhost:3000/users

App will display all of the users in the table.

http://localhost:3000/users/1

App will display the first user in the table, or the user with id placed where the 1 is.

http://localhost:3000/users/999999

App will display error message because user is not present in the table.
